# AI Visibility Report Generator — Thrive Agency Edition

White-labeled version of the AI Visibility Report Generator, branded for Thrive Agency.

## Brand Colors
- **Primary Green:** `#7D963D` (Thrive brand green)
- **Light Green:** `#D7F0A3` (accent)
- **Orange CTA:** `#FF6600` (buttons & UI accents)
- **Report Header:** Dark green gradient `#1a2e1a → #2d5e2d`

## Architecture
- `server.js` — Express 4 proxy server
- `public/index.html` — Full frontend with embedded CSS/JS
- Deployed on Render.com (auto-deploys on git push)

## Setup
```bash
npm install
npm start
# Visit http://localhost:3000
```
